#### 停用 Finder 提示通知的 iOS Update
```shell
# 雖然 10.15 後沒有 iTunes 但還是用這服務來看設定值 （要重開機）
defaults write com.apple.itunes disableCheckForDeviceUpdates -bool YES
```
