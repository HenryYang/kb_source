## 常見符號英文

* `/` -> slash

* `#` -> sharp/hashtag/number

* `-` -> hyphen

* `—` -> dash

* `_` -> underline

* `&` -> ampersand/and

* `!` -> bang or exclamation

* `?` -> question mark

* `\` -> backslash

* `*` -> star symbol

* `~` -> tilde

* `()` -> brackets

* `*` -> asterisk

* `’` -> apostrophe

* `:` -> colon

* `;` -> semi-colon

* `‘ ’` -> single quotation mark 

* `“ ”` -> double quotation mark

* `()` -> parentheses/ round brackets

* `[]` -> brackets/ square brackets

* `{}` ->  braces/ curly brackets

* `<>` -> angle brackets

* `,` -> comma
